# Dummy-Project-Sistem-Reservasi-Kursi-Bioskop

# Deskripsi Proyek
Proyek ini adalah sebuah simulasi sistem reservasi kursi bioskop berbasis Python yang dirancang untuk mengelola alur pemesanan tiket secara efisien. Menggabungkan logika backend yang kuat dengan interaksi pengguna yang intuitif, program ini mendemonstrasikan pemahaman mendalam tentang manajemen struktur data, kontrol alur logika, dan penanganan transaksi operasional. Fokus utama proyek ini terletak pada keakuratan pemrosesan data pesanan dan pencegahan anomali input.

# Fitur Utama
- Visualisasi Pemetaan Kursi Dinamis: Menyajikan tata letak matriks bioskop secara real-time, memungkinkan sistem melacak dan memperbarui status ketersediaan kursi secara otomatis.
- Sistem Validasi Input Ketat: Mengimplementasikan pembatasan karakter dan pengecekan tipe data pada input pengguna untuk memastikan integritas data transaksi tetap terjaga tanpa error.
- Manajemen Transaksi & Kalkulasi: Memproses perhitungan total harga pemesanan secara presisi layaknya sistem kasir profesional, lengkap dengan konfirmasi akhir. (Admin Only)
- Pembaruan Status Otomatis (State Management): Mengamankan kursi yang telah dipesan dengan mengubah status data ketersediaan seketika setelah pembayaran atau konfirmasi berhasil divalidasi.

# Teknologi & Keterampilan yang Ditonjolkan
- Bahasa Pemrograman: Python
- Fokus Keterampilan: Computational thinking, manipulasi struktur data (arrays/lists), implementasi kontrol alur bersyarat, dan pembersihan/validasi data.

# Nilai Tambah & Pembelajaran
Pengembangan sistem ini menjadi bukti kemampuan dalam merancang arsitektur perangkat lunak yang terstruktur, modular, dan tahan terhadap edge cases. Pengelolaan data ketersediaan kursi dan pembuatan alur transaksi ini membangun fondasi algoritmik yang kuat—sebuah keahlian krusial untuk pemrosesan data berskala lebih besar dan pemecahan masalah operasional bisnis di dunia nyata.
